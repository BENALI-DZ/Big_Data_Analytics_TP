Install Apache Kafka
# By Docker 

1/ Run docker 

2/ Create Folder D:\Master_TP\BIGDATA\Kafka\

3/ then create file docker-compose.yml  from this folder

4/ Open Windows Powershell and goto this folder and run command : docker-compose up -d
